# Rock Paper Scissors Game

A simple and interactive **Rock Paper Scissors** game built using **HTML, CSS, and JavaScript**. Play against the computer, track your score, and even enable auto-play mode!

---

## Features

- Play Rock, Paper, Scissors with one click  
- Computer generates random moves  
- Score tracking (Wins, Losses, Ties)  
- Score saved using **localStorage**  
- Reset score option  
- Auto-play mode  

---

## Demo

Click buttons:
- ✊ Rock  
- 🖐️ Paper  
- ✌️ Scissors  

Game result and score will update instantly on screen.

## How It Works

- User clicks a button (rock/paper/scissors)
- JavaScript function `play()` is called
- Computer randomly selects a move
- Winner is determined based on rules:
  - Rock beats Scissors  
  - Scissors beats Paper  
  - Paper beats Rock  
- Score updates and is saved in browser storage
